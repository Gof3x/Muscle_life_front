# Muscle Life Supplements - Vitrine Frontend

Projeto frontend de vitrine para a loja "Muscle Life Supplements" criado com Vite + React + TypeScript + Tailwind CSS.

Principais comandos:

- Instalar dependências:

```powershell
npm install
```

- Executar em desenvolvimento:

```powershell
npm run dev
```

O projeto é 100% frontend (sem backend). Produtos estão em `src/data/products.ts` e o estado do carrinho usa `src/context/CartContext.tsx`.
